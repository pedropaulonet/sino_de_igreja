🔔 Sino de Igreja

Programa para tocar sinos em horários programados, ideal para igrejas e Raspberry Pi.

Interface Terminal (curses):

📦 Instalação

Pré-requisitos:

  • Python 3.6+
  • pygame (opcional mas recomendado)
  • Arquivos de áudio MP3
pip install pygame

Se pygame não estiver disponível, o programa usa automaticamente: ffplay (ffmpeg), paplay (PulseAudio), aplay (ALSA) ou omxplayer (Raspberry Pi).

⚙️ Configuração

Edite o arquivo config.json para configurar horários e sons. Exemplo mínimo:

{
  "sons": {
    "sino": "sounds/sino.mp3",
    "nossa_senhora": "sounds/nossa_senhora.mp3"
  },
  "programacao": [
    {"hora": 7, "minuto": 0, "som": "sino", "repeticoes": 7},
    {"hora": 19, "minuto": 0, "som": "nossa_senhora", "repeticoes": 1}
  ]
}

Campos do config.json:

  • sons - Mapeia nomes (chaves) para arquivos MP3 em sounds/.
  • programacao - Lista de toques (cada item é um horário).
  • hora - Hora do toque (0-23).
  • minuto - Minuto do toque (0-59).
  • som - Chave do som definido em sons.
  • repeticoes - Quantas vezes o som será reproduzido.

🚀 Como Executar

Interface Gráfica (Tkinter) - modo janela:

./sino_igreja_gui.sh

ou

python3 sino_igreja_gui.py

Interface Terminal (curses) - modo gráfico no terminal:

./sino_igreja.sh

ou

python3 sino_igreja.py

Modo console (sem interface):

python3 sino_igreja.py --console

📋 Ver informações do programa

python3 sino_igreja.py --sobre
